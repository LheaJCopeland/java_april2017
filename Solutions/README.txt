Midterm Project example solution.
The project contains examples of the development of a Point of Sale application.

1. pos5Gui - contains a Java Swing version of the pos application developed by reusing the core classes in the pos console.

Running the jar file.
If you have Java installed on your machine you can run the application executable jar files with the command using
java -jar filename.jar on the terminal/command line.

pos5Gui.jar is too large to push to the github repo and is provided upon request.
